D3 in HTML

The data used to create the charts used in this assignment is taken from World Bank's data on Rural Population. The dataset provides information in the year range 1960-2017 for 264 countries, but the charts shown below are built using 2017 data on 20 countries only for convenience. The charts and tables used in this web page are built using D3 Version 5.

Rural Population Data (Table)

As mentioned above, the data used is for Rural Population in 20 countries. The countries with the largest rural populations were chosen to compare the variation in such a short range. These countries, with their country code and rural population (as % of total population) in 2017 are shown in the table below.

Table 1 Country name, code and percentage rural population for 20 countries with the largest rural population

Rural Population Data (Bar Chart)

The data shown in Table 1 was used to create the bar chart in this section. The colors of each bar are determined by the magnitude of their value (the darker the color, the greater the rural population). The values on the X-axis show the country code and the values on Y axis show rural population (as % of total population).

Figure 1 Bar chart showing Rural Population (as % of Total Population) in 2017 for 20 Countries

Rural Population Data (Scatterplot)

The data from Table 1 was used to create the scatterplot shown in Figure 2. Compared to the bar chart in Figure 1, the scatterplot is tougher to read and difficult to compare values with. The color scheme used in Figure 1 was also used here. The values on the X-axis show the country code and the values on Y axis show rural population (as % of total population).

Figure 2 Scatterplot showing Rural Population (as % of Total Population) in 2017 for 20 Countries

Rural Population Data (Slope graph)

The data used to construct this graph used a different dataset that the rest of the charts on this page. An additional column was added to the data in Table 1 to show how the rural population changed between the years 1960 and 2017. The issue faced with the slopegraph is that values overlap when they are close to each other on scale. In this case, there were overlapping values even when the height of the svg chart was set to 16000px. To avoid this, only 10 countries were chosen such that the information is clearly displayed. Using the slopegraph, we can observe that in most of the countries rural population has reduced.

Figure 3 Slope graph showing Rural Population (as % of Total Population) change from 1960 to 2017 for 20 Countries
