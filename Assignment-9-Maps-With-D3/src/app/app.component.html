Assignment 8: Responsive Charts

The data used for generating the charts shown below is taken from the UN Database for GDP of countries and the County of Los Angeles Open Data and was modified to focus on the top 10 countries with the largest economies. They are:
  • Brazil
  • Canada
  • China
  • France
  • Germany
  • India
  • Italy
  • Japan
  • United Kingdom
  • United States of America

1) World Proportional Symbol Map

The following chart shows a circle for each of the countries mentioned above with its radius relative to the GDP of that country in the year 2016. Using the radius of circles given in the legend, each country's GDP can be calculated accurately using a scale or can be estimated intuitively.

Proportional Symbol Map showing GDP for top 10 countries with the largest economies



2) World Chloropleth Map

The following chart colour codes each country with the colour intensity relative to the GDP of the country. The colour corresponds to the legend, where each colour is a threshold. Here, the value cannot be estimated accurately by any scale, only intuitively.

Chloropleth Map showing GDP for top 10 countries with the largest economies



3) Los Angeles County Chloropleth Map

The following chart colour codes each neighborhood in the Los Angeles County with the colour intensity relative to the number of museums in the neighborhood, similar to the previous chart. Since there are a large number of labels in this chart, they are shown using a tooltip which becomes visible when hovered upon it. This tooltip also shows the number of museums so that they don't have to be estimated intuitively.

Chloropleth Map showing number of museums in the neighborhoods of the Los Angeles County
