Information Visualization Critiquing

The infographics chosen for this assignment provide information on water crisis around the world. The first infographic is taken from UNICEF USA which summarizes the water crisis around the world in 2014.

The second infographic is taken from Huffington Post and it identifies and summarizes the five main drivers (according to the creator of the infographic) of water crisis. These infographics are analysed using a Visualization Wheel as it can be seen from the sections that follow.

Visualization Wheels

A visualization wheel consists of 6 axes along which infographics can be compared. They are:
  • Abstraction - Figuration
  • Functionality - Decoration
  • Density - Lightness
  • Multidimensionality - Unidimensionality
  • Originality - Familiarity
  • Novelty - Redundancy
The two infographics shown above are compared in the table below using the visualization wheels generated.




Visualization Wheel Axis UNICEF Infographic Huffington Post Infographic
Abstraction - Figuration This infographic contains a little abstraction (shown as water containers and a glass) and has high figuration as the data is shown on a world map which bears high resemblance with the object in reality This infographic contains a lot of abstraction as each figure is represented by abstract versions of a real object and has very low figuration as there are no images that resemble the objects that they wish to explain
Functionality - Decoration This infographic contains higher functionality (choropleth and lollipop chart) and a small amount of decoration (random water container and a glass) This infographic contains a lower amount of functionality and a very large amount of decoration (unnecessary water droplets)
Density - Lightness This infographic is relatively dense, however it provides a significant amount of information This information is very light with a lot of space left empty, however it also provides a lot of information
Multidimensionality - Unidimensionality This infographic can be considered multidimensional because of the choropleth used in it This infographic can be considered more unidimensional as the figures used are relatively simple and contain singular dimensional information and in a few cases, two-dimensional
Originality - Familiarity This infographic contains familiar charts (choropleth, lollipop chart, single numeric field) and not many new charts This infographic contains a lot of customized charts made for their own purpose, making it more original and less familiar
Novelty - Redundancy This infographic covers a lot of different topics without any overlap among them, leading to a high novelty This infographic talks about population growth and middle class growth, two topics which are similar, creating a slight redundancy

Potential Improvements

Based on the analysis made using the visualization wheel, the following improvements can be made to the infographics shown above.

Improvements in UNICEF infographic

  • Abstractions used to make the infographic look better can be improved as the water container with the glass seems randomly placed
  • Single numeric values can be merged to be more informative, thus making the chart more multidimensional
  • Decorative items can be added to the infographic in places which require more attention, moving the readers focus there

Improvements in Huffington Post infographic

  • The transitions between each cause can be adjusted to form a grid (or matrix) to reduce the lightness of the infographic
  • Unnecessary and meaningless decorations (Example: image depicting climate change) can be replaced with real world functional items to make the user understand and relate better to the message conveyed
  • Redundancy (mentioned in the description of the visualization wheel) can be reduced in the infographic by understanding the topic better and recomputing the number of causes
  • A few familiar charts can be introduced to reduce the effort for the reader by making the infographic simpler
